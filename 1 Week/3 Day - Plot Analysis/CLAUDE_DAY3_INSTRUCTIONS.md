## 4. Откуда брать числа (переиспользуй логику Day 2)

Не изобретай заново — повтори расчётные блоки из `Metro_Traffic_EDA.ipynb`. Ключевые куски:

**Загрузка + чистка + честная почасовая сборка** (ячейки 2 и 17 старого ноутбука):
```python
df = pd.read_csv('Metro_Interstate_Traffic_Volume.csv')
df['date_time'] = pd.to_datetime(df['date_time'], format='%d-%m-%Y %H:%M')

clean = df.copy()
clean.loc[clean['temp']==0, 'temp'] = np.nan
clean.loc[clean['rain_1h']>350, 'rain_1h'] = np.nan

hourly = clean.groupby('date_time').agg(
    traffic=('traffic_volume','first'),
    is_snow=('weather_main', lambda x:(x=='Snow').any()),
).reset_index()
hourly['hour'] = hourly['date_time'].dt.hour
hourly['month'] = hourly['date_time'].dt.month
hourly['is_weekend'] = hourly['date_time'].dt.weekday>=5
```

**График 1 — часы** (из ячейки 20): `hourly.groupby('hour')['traffic'].agg(['mean','count'])`.
Пик ≈ 5 709 в 16:00, минимум ≈ 373 в 3:00. Аннотируй обе точки.

**График 2 — будни/выходные** (из ячейки 22): `hourly.groupby('is_weekend')['traffic'].mean()`.
Будни 3 557, выходные 2 624 → **+35.6%**. Подпиши процент стрелкой между столбцами.

**График 3 — праздник vs обычные (зацепка)** (из ячейки 26):
```python
hol = clean['holiday'].notna()
clean.groupby(hol)['traffic_volume'].agg(['mean','count'])
# Праздничный час ≈ 865 маш/ч (61 строка), обычные ≈ 3000+
```
Покажи как два столбца — праздник выглядит провально низким. **Намеренно** без контекста, чтобы создать «ловушку».

**График 4 — где стоит метка holiday (поворот)**:
```python
sorted(clean[hol]['date_time'].dt.hour.unique())  # → [0] только полночь
```
Покажи 24 часовых слота, подсветь цветом единственный (00:00), где есть метки; остальные 23 — серым/пустым. Подпись: «`holiday` живёт только в 00:00».

**График 5 — честное сравнение (челюсть)**:
```python
midnight_all = clean[clean['date_time'].dt.hour==0]['traffic_volume'].mean()  # ≈ 835
# три столбца: "Праздничный час (865)" / "Любая полночь (835)" / "Средние сутки (~3260)"
```
Вывод визуально: 865 ≈ 835 → праздник неотличим от обычной ночи. Разница с «сутками» — это про время, а не праздник.

## 5. Требования к дизайну графиков (научно + красиво)

Это самое важное для защиты — графики должны быть «вкусными» и читаться за 2 секунды.

**Единый стиль — задай один раз в ячейке 2:**
```python
import matplotlib.pyplot as plt
import matplotlib.ticker as mticker
import numpy as np, pandas as pd

plt.rcParams.update({
    'figure.dpi': 130,
    'figure.figsize': (10, 5.5),
    'font.size': 12,
    'axes.titlesize': 15,
    'axes.titleweight': 'bold',
    'axes.spines.top': False,
    'axes.spines.right': False,
    'axes.grid': True,
    'grid.alpha': 0.25,
    'axes.axisbelow': True,
})
# палитра: один акцент, остальное приглушено
INK   = '#1a1a2e'   # тёмный текст/линии
MUTED = '#c8ccd4'   # серый «фон-данные»
ACCENT= '#e63946'   # красный — указатель на главное
COOL  = '#457b9d'   # спокойный синий для нейтральных серий
```

**Жёсткие правила (из задания):**
- **Заголовок = вывод.** Используй `ax.set_title(...)` с выводом. Тему/детали — в подпись оси или в маленький `ax.text` сверху серым.
- **Ось Y от нуля** на всех bar-графиках. Никакого обрезания для драматизма.
- **Цвет со смыслом:** выделяй ОДНО ключевое значение `ACCENT`-цветом, всё остальное — `MUTED`/`COOL`. Цвет — указатель, не раскраска.
- **Аннотируй прямо на графике:** пик, минимум, % разницы, «ловушку» — стрелкой (`ax.annotate(..., arrowprops=dict(arrowstyle='->'))`) или подписью значения над столбцом. Не заставляй читателя искать.
- Подписывай значения над столбцами: `ax.bar_label` или `ax.text`.
- Каждый график: `plt.tight_layout()` и достаточно крупный шрифт (видно с проектора за 5 м).
- Где уместно — лёгкий приём поверх диаграммы: затенённая зона пиковых часов (`axvspan`), пунктирная линия среднего (`axhline`), вынос-аннотация. Сдержанно, не перегружая.

**Конкретика по графикам:**
- **График 1 (часы):** линия с маркерами; залей `axvspan` утренний (7–9) и вечерний (16–18) пики бледным акцентом; аннотируй пик 16:00 и минимум 3:00 стрелками с числами.
- **График 2 (будни/выходные):** 2 столбца, будни — `ACCENT`, выходные — `MUTED`; между ними фигурная скобка/стрелка с «+35.6%».
- **График 3 (зацепка):** 2 столбца, «праздник» подсвечен тревожно-низким; подпись «выглядит убедительно… но так ли это?» — задаёт крючок.
- **График 4 (поворот):** 24 ячейки/бара по часам, только 00:00 — `ACCENT`, остальные серые/пустые; крупная подпись «метка только в полночь → сравнение нечестное».
- **График 5 (челюсть):** 3 столбца; первые два («Праздничный час», «Любая полночь») почти равны — соедини их пунктиром с подписью «≈ одно и то же»; третий («Средние сутки») высокий `COOL`. Заголовок-вывод: «Праздник неотличим от обычной ночи — пустела не дорога, а часы».
